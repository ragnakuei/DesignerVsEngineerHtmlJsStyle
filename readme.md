### 功能：

- 點擊在清單資料上的各筆資料，會 alert 顯示該筆資料

### 強調：

- 不要把事件註冊，寫死在 onload 中

- 由於後端工程師需要加工 (例如：改成需要從 ajax 來更新資料)，因此所以建議設計師，一開始就先把事件註冊的語法放至 function 中去呼叫 !

- 這個 style 不會對設計師造成負擔 !
